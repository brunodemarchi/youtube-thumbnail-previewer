# YouTube Thumbnail Previewer

Preview how your YouTube thumbnail looks in context — alongside real video thumbnails in YouTube's actual UI layouts.

## Features

- **Three preview modes**: Home Grid, Sidebar (Up Next), and Search Results
- **Upload your thumbnail**: click, drag & drop, or paste (Ctrl+V) anywhere on the page
- **Edit video details**: title, duration (auto-formatted), and channel name
- **Randomize**: shuffle surrounding thumbnails and your video's position in the layout
- **30 real YouTube thumbnails** bundled locally for instant, offline-ready previewing
- **Persistent**: your thumbnail and settings are saved in localStorage across sessions
- **Fully local**: no backend, no data sent anywhere

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite
- No external UI libraries
